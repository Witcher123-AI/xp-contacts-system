# 通讯录管理系统

一个基于原生HTML、CSS和JavaScript开发的现代化通讯录管理系统，支持联系人的增删改查、收藏、多种联系方式管理以及Excel导入导出功能。

## 功能特性

### 1. 核心功能
- **添加联系人**：支持添加姓名、多种联系方式、备注等信息
- **编辑联系人**：可随时修改联系人的所有信息
- **删除联系人**：删除不需要的联系人
- **查看详情**：点击联系人查看详细信息

### 2. 收藏功能
- **收藏联系人**：标记重要联系人
- **收藏筛选**：快速查看所有收藏的联系人

### 3. 多种联系方式
- **支持的方式**：电话、邮箱、微信、地址
- **灵活添加**：可添加多个同类型联系方式
- **自由删除**：可删除不需要的联系方式

### 4. Excel导入导出
- **导出功能**：将联系人列表导出为Excel文件
- **导入功能**：从Excel文件批量导入联系人
- **格式支持**：兼容标准Excel格式(.xlsx)

### 5. 界面设计
- **响应式布局**：适配不同屏幕尺寸
- **简洁美观**：现代化的UI设计
- **交互友好**：流畅的操作体验
- **实时搜索**：快速查找联系人
- **标签筛选**：按条件筛选联系人

## 技术栈

- **前端框架**：原生HTML5、CSS3、JavaScript(ES6+)
- **数据存储**：localStorage本地存储
- **Excel处理**：SheetJS库(https://sheetjs.com/)
- **样式设计**：CSS Flexbox、CSS Grid
- **模块化结构**：ES6 Modules

## 文件说明

### 1. 主文件
- **index.html**：应用的入口文件，包含页面结构和模块引入

### 2. 样式文件
- **base.css**：全局样式、重置样式和基础组件样式
- **sidebar.css**：侧边栏、联系人列表和筛选控件样式
- **content.css**：联系人详情展示和操作按钮样式
- **modal.css**：添加/编辑联系人模态框样式

### 3. 脚本文件
- **app.js**：主应用逻辑，包含DOM操作、事件处理和界面渲染

### 4. 数据模型
- **Contact.js**：联系人数据模型，定义联系人的属性和方法

### 5. 服务层
- **contactService.js**：处理联系人数据的增删改查操作
- **storageService.js**：封装localStorage的存储操作
- **importExportService.js**：处理Excel文件的导入导出功能

## 安装和使用

### 本地运行
1. 克隆或下载项目文件
2. 直接用浏览器打开`index.html`文件即可使用

### 功能使用

#### 添加联系人
1. 点击"添加联系人"按钮
2. 填写联系人信息（姓名、联系方式、备注等）
3. 可选：勾选"设为收藏"
4. 点击"保存"按钮完成添加

#### 编辑联系人
1. 点击联系人列表中的联系人查看详情
2. 在详情页点击"编辑"按钮
3. 修改联系人信息
4. 点击"保存"按钮完成修改

#### 删除联系人
1. 点击联系人列表中的联系人查看详情
2. 在详情页点击"删除"按钮
3. 确认删除操作

#### 收藏联系人
1. 在联系人详情页点击"收藏"按钮
2. 可在"收藏"标签页查看所有收藏的联系人

#### 导入/导出联系人
1. 导出：点击"导出"按钮，联系人列表将导出为Excel文件
2. 导入：点击"导入"按钮，选择Excel文件，联系人将批量导入系统

## 数据存储

应用使用浏览器的localStorage进行数据存储，所有联系人数据保存在本地，不会上传到服务器。

**注意**：清除浏览器缓存或使用隐私模式可能会导致数据丢失。
